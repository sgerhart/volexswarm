# VolexSwarm

A sophisticated **multi-agent AI trading system** built on an agentic AI framework that transforms traditional FastAPI services into intelligent, autonomous AutoGen agents.

## 🚀 **Quick Start**

```bash
# Start the system
docker-compose up -d

# Check status
docker-compose ps

# View logs
docker-compose logs -f
```
